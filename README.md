# Accredian Enterprise Landing Page Clone

## 🔗 Live Demo

https://accredian-clone-omega.vercel.app/

## 📌 Project Overview

This project is a partial clone of the Accredian Enterprise landing page, built using Next.js. The goal was to recreate the core UI and structure of the website with a focus on responsiveness, clean design, and reusable components.

## 🚀 Tech Stack

* Next.js (App Router)
* React (Functional Components & Hooks)
* Tailwind CSS
* Vercel (Deployment)

## ✨ Features

* Fully responsive design (mobile + desktop)
* Reusable and modular components
* Clean and structured layout
* Smooth navigation across sections
* Optimized for readability and maintainability

## 📂 Project Structure

app/ – Main application routes (App Router)
components/ – Reusable UI components (Navbar, Hero, Sections, Footer)
public/ – Static assets
styles/ – Global styles

## ⚙️ Setup Instructions

```bash
git clone https://github.com/mdshoaibkhan/accredian-clone.git
cd accredian-clone
npm install
npm run dev
```

## 🧠 Approach

* Broke the UI into smaller reusable components such as Navbar, Hero section, and Footer
* Used Tailwind CSS for fast and consistent styling
* Focused on responsiveness using flexbox, grid, and Tailwind breakpoints
* Prioritized clean structure and maintainable code over pixel-perfect design

## 🤖 AI Usage

AI tools were used to improve development speed and efficiency:

* Generated base component structures
* Assisted with Tailwind styling and layout suggestions
* Helped debug minor UI and responsiveness issues

Manual work included:

* Structuring the component hierarchy
* Refining layout and spacing
